# 📚 КурсПлатформа — Flask + PostgreSQL

Система управления учебными курсами студентов с авторизацией, тремя ролями и расширенной статистикой.

---

## 🚀 Быстрый старт

### 1. Установить зависимости
```bash
pip install -r requirements.txt
```

### 2. Создать базу данных PostgreSQL
```sql
-- В psql или pgAdmin:
CREATE DATABASE courses_db;
```

### 3. Инициализировать схему
```bash
psql -U postgres -d courses_db -f init_db.sql
```

### 4. (Опционально) Заполнить демо-данными
```bash
python seed_demo.py
```

### 5. Запустить приложение
```bash
python app.py
```

Приложение будет доступно по адресу: **http://localhost:5000**

---

## ⚙️ Настройка подключения к БД

Через переменные окружения (или измените `DB_CONFIG` в `app.py`):

```bash
export DB_HOST=localhost
export DB_PORT=5432
export DB_NAME=courses_db
export DB_USER=postgres
export DB_PASSWORD=postgres
export SECRET_KEY=your-secret-key
```

## 🔐 Роли и права доступа

### 👑 Администратор
- Полное управление пользователями (добавить, удалить)
- Управление курсами (создать, редактировать, удалить)
- Просмотр и редактирование всех студентов и преподавателей
- Выставление оценок
- Полная статистика по всем разделам

### 🎓 Преподаватель
- Просмотр своих курсов и студентов
- Выставление оценок
- Статистика

### 🎒 Студент
- Запись на курсы
- Просмотр своих курсов
- Просмотр своих оценок

---

## 📈 Статистика

Страница статистики содержит 4 вкладки:

| Вкладка | Показатели |
|---------|-----------|
| Студенты | Всего, активных, ср./макс./мин. балл, топ-5 |
| Курсы | Всего, с преподавателем, ср. студентов, популярные |
| Преподаватели | Всего, ведут курсы, ср. курсов, топ-5 |
| Оценки | Распределение (отл./хор./удовл./неудовл.) + диаграмма |

---

## 🗄️ Схема базы данных

```
users ─────────────┬── teachers ──── courses ──── enrollments ──── grades
                   └── students ─────────────────────────┘
```

- **users** — все пользователи (admin, teacher, student)
- **teachers** — расширение профиля преподавателя
- **students** — расширение профиля студента
- **courses** — учебные курсы
- **enrollments** — записи студентов на курсы
- **grades** — оценки за курсы
