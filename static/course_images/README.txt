Картинки курсов хранятся здесь и не требуют поля в БД.

Как это работает:
- приложение подставляет картинку по названию курса;
- если название совпадает с одним из стандартных курсов, берётся соответствующий .svg;
- если совпадения нет, используется default.svg.

Файлы по умолчанию:
- python.svg — Основы Python
- sql.svg — Базы данных и SQL
- flask.svg — Веб-разработка на Flask
- math_analysis.svg — Математический анализ
- linear_algebra.svg — Линейная алгебра
- ml.svg — Машинное обучение
- economy.svg — Экономика предприятия
- project_management.svg — Управление проектами
- discrete_math.svg — Дискретная математика
- algorithms.svg — Алгоритмы и структуры данных
- default.svg — запасная картинка

Можно просто заменить эти файлы своими изображениями, сохранив те же имена.
